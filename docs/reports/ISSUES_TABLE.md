## 边界条件检查清单

### 输入验证边界

- [ ] 空字符串输入
- [ ] 超长字符串 (>10000 字符)
- [ ] 特殊字符 (`$`, `` ` ``, `\`, `"`, `'`)
- [ ] Unicode/Emoji
- [ ] 换行符和控制字符
- [ ] 路径遍历 (`../`, `../../`)
- [ ] Windows 保留文件名 (`CON`, `NUL`, `PRN`)

### 文件系统边界

- [ ] 磁盘空间不足
- [ ] 文件不存在
- [ ] 目录不存在
- [ ] 无读权限
- [ ] 无写权限
- [ ] 只读文件系统
- [ ] 符号链接循环
- [ ] 文件名过长 (>255 字节)
- [ ] 路径过长 (>4096 字节)

### 并发边界

- [ ] 多用户同时运行
- [ ] 同一用户多实例
- [ ] 文件被其他进程锁定
- [ ] 配置文件同时修改

### 环境边界

- [ ] `$HOME` 未设置
- [ ] `$TERM` 未设置
- [ ] `$PATH` 损坏
- [ ] `$LANG`/`$LC_*` 非 UTF-8
- [ ] Bash 版本 < 4.0
- [ ] 依赖命令缺失

### 数据边界

- [ ] 空文件
- [ ] 格式损坏的 Markdown
- [ ] 缺少必需的章节
- [ ] 重复的 prompt 标题
- [ ] 超大文件 (>100MB)
